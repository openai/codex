世界中の開発者が**母国語でコードレビュー**を受けられるようになったで！

#### 対応言語（8言語）

1. 🇯🇵 **日本語 (ja)** - 完全対応
2. 🇬🇧 **英語 (en)** - 完全対応

4. 🇰🇷 **韓国語 (ko)** - 完全対応
5. 🇫🇷 **フランス語 (fr)** - 完全対応
6. 🇩🇪 **ドイツ語 (de)** - 完全対応
7. 🇪🇸 **スペイン語 (es)** - 完全対応
8. 🇵🇹 **ポルトガル語 (pt)** - 完全対応

#### 使用方法（超簡単！）

**AGENTS.md**に1行追加するだけ：
```markdown
- language: ja
```

**コマンド実行**：
```bash
codex
> /review src/main.rs

🔍 コードレビュー結果
（日本語でレビューが表示される！）
```
