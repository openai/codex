## ✨ 新機能の強調

### 1. **自律オーケストレーション**

```
TaskAnalyzer → AutoOrchestrator → Session Manager
     ↓              ↓                  ↓
  分析結果      自動委譲          実行管理
```

- 複雑度が0.7を超えると自動的にサブエージェントを起動
- ClaudeCode風の自律的な動作
- ユーザーの介入不要

### 2. **7つの専門サブエージェント**

| エージェント | 役割 | トークン予算 |
|------------|------|------------|
| CodeExpert | コード分析・リファクタリング | 40,000 |
| SecurityExpert | セキュリティ監査 | 50,000 |
| TestingExpert | テスト生成 | 30,000 |
| DeepResearcher | ディープリサーチ | 60,000 |
| DocsExpert | ドキュメント生成 | 30,000 |
| DebugExpert | デバッグ | 40,000 |
| PerformanceExpert | パフォーマンス最適化 | 40,000 |

### 3. **ディープリサーチエンジン**

- **多元ソース**: DuckDuckGo, Brave, Google, Bing
- **引用付き**: すべての情報にソース帰属
- **矛盾検出**: 競合する情報を自動識別
- **深度設定**: 1-5レベルの調査深度
- **信頼性スコア**: 各発見の信頼性メトリクス

## 🔧 技術的詳細

### アーキテクチャ図の特徴

1. **4層構造**
   - **UI層**: CLI/TUI
   - **ランタイム層**: Core Runtime
   - **エージェント層**: Sub-Agents
   - **プロバイダー層**: LLM APIs & Tools

2. **データフロー**
   - ユーザー入力 → TaskAnalyzer
   - 分析結果 → AutoOrchestrator
   - 委譲判定 → Sub-Agents
   - 実行結果 → Session Manager

3. **コンポーネント間連携**
   - Supervisor: サブエージェント管理
   - Deep Research: 調査エンジン
   - Custom Commands: カスタムコマンド実行

```bash

codex

# サブエージェント委譲
codex delegate code-reviewer --scope ./src

codex research "React Server Componentsのベストプラクティス" --depth 3

# 並列実行
codex delegate-parallel code-reviewer,test-gen --scopes ./src,./tests
```
