# Slash Commands 使用说明

> 本文是 `docs/slash_commands.md` 的中文概览版本。更详细的例子和配置格式请参考英文原文。

Slash Commands 是在 Codex 中快速调用预定义动作或提示模版的一种方式。

## 基本用法

- 在输入框中键入 `/`，Codex 会展示可用命令列表。
- 继续输入可以过滤列表，例如 `/sec` 过滤出 `/security_review`。
- 选择后：
  - 有的命令会将模版插入输入框，方便你再编辑。
  - 有的命令可能直接执行相应操作。

## 在配置中定义命令

你可以在 `config.toml` 中：

- 定义命令名（如 `/review`）。
- 绑定到某段提示模版。
- 声明参数占位符（例如需要用户额外输入文件名、PR 链接等）。

示例（伪代码，字段以英文文档为准）：

```toml
[slash_commands.review]
prompt = "请审核当前 diff 并给出改进建议。"
```

## 场景示例

- 代码审查流程。
- 安全检查与报告。
- 生成某类文档（变更说明、API 文档等）。
- 快速运行某种常用重构或迁移工作流。
