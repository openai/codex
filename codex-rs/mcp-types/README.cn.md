# mcp-types（中文说明）

Model Context Protocol 的类型定义。灵感来源于 <https://crates.io/crates/lsp-types>。

正如规范 <https://modelcontextprotocol.io/specification/2025-06-18/basic> 所述：

- TypeScript schema 是权威来源：<https://github.com/modelcontextprotocol/modelcontextprotocol/blob/main/schema/2025-06-18/schema.ts>
- JSON Schema 适合自动化工具使用：<https://github.com/modelcontextprotocol/modelcontextprotocol/blob/main/schema/2025-06-18/schema.json>
