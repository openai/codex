# @openai/codex-responses-api-proxy（中文说明）

```shell
npm i -g @openai/codex-responses-api-proxy
```

通过上述命令可以安装预构建的 `codex-responses-api-proxy` 可执行文件。本 npm 包会为 macOS、Linux 和 Windows 分发对应平台的二进制（详见源码仓库中的 [Codex Responses API proxy](https://github.com/openai/codex/tree/main/codex-rs/responses-api-proxy)）。

查看可用选项：

```shell
node ./bin/codex-responses-api-proxy.js --help
```

更详细的说明请参考 [`codex-rs/responses-api-proxy/README.md`](https://github.com/openai/codex/blob/main/codex-rs/responses-api-proxy/README.md)。
